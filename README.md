🌸 Feliz Día de la Madre

Proyecto web interactivo inspirado en el Día de la Madre, desarrollado con el objetivo
de combinar creatividad, diseño web y programación front-end en una experiencia visual emotiva 
e interactiva.

✨ Descripción
La página presenta animaciones, efectos visuales y elementos dinámicos diseñados para 
transmitir un mensaje especial de una manera moderna y creativa.  El proyecto fue realizado
aplicando buenas prácticas de estructura web y diseño responsivo.

🚀 Funcionalidades
- Animaciones suaves e interactivas
- Efectos visuales personalizados
- Corazones y elementos dinámicos en pantalla
- Diseño adaptable a diferentes dispositivos
- Interacción mediante JavaScript
- Interfaz moderna y atractiva

🛠️ Tecnologías utilizadas

- HTML
- CSS
- JavaScript

📚 Objetivo del proyecto

Fortalecer habilidades en desarrollo web front-end mediante la creación de una experiencia interactiva 
enfocada en animaciones, diseño visual y manipulación de elementos dinámicos.

💻 Desarrollo

El proyecto fue desarrollado en Visual Studio Code y posteriormente publicado en GitHub como parte de
mi portafolio de proyectos personales y académicos.
